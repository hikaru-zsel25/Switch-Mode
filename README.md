# 🎯 Switch Mode アプリ 学習チュートリアル

このプロジェクトは、HTMLの基礎を学びながら実用的なWebアプリを作成できるチュートリアルです。

## 📋 学習目標

- セマンティックなHTMLの書き方を学ぶ
- IDとクラスの適切な使い分けを理解する
- レスポンシブデザインの基礎を体験する
- JavaScriptとHTMLの連携方法を学ぶ

## 🚀 始め方

1. `index.html`を開く
2. コメントに従って段階的にHTMLを記述する
3. 各ステップで保存してブラウザで確認する
4. 完成したら`index-complete.html`と比較する

## 📝 学習のポイント

### HTMLの基本構造
```html
<!doctype html>
<html lang="ja">
<head>
  <!-- メタデータ -->
</head>
<body>
  <!-- コンテンツ -->
</body>
</html>
```

### セマンティックタグの使用
- `<header>` - ヘッダー情報
- `<main>` - メインコンテンツ
- `<footer>` - フッター情報
- `<section>` - セクション区切り

### IDとクラスの使い分け
- **ID**: 一意な要素（JavaScriptで操作する要素）
- **クラス**: 複数の要素に適用するスタイル

### レスポンシブデザイン
- `viewport` メタタグの設定
- CSS Gridとメディアクエリの活用

- [ ] ブラウザで正常に表示されるか

## 💡 よくある間違い

1. **タグの閉じ忘れ**
   ```html
   <!-- ❌ 間違い -->
   <div>
   <p>テキスト
   
   <!-- ✅ 正しい -->
   <div>
     <p>テキスト</p>
   </div>
   ```

2. **IDの重複**
   ```html
   <!-- ❌ 間違い -->
   <div id="button">ボタン1</div>
   <div id="button">ボタン2</div>
   
   <!-- ✅ 正しい -->
   <div id="button1">ボタン1</div>
   <div id="button2">ボタン2</div>
   ```

3. **属性の記述間違い**
   ```html
   <!-- ❌ 間違い -->
   <input type=text id=name>
   
   <!-- ✅ 正しい -->
   <input type="text" id="name">
   ```
